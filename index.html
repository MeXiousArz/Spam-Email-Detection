Spam Email Detection

Machine Learning Project using Naive Bayes

Project Overview

This project focuses on detecting spam emails using Naive Bayes classification techniques. Different text vectorization approaches such as Bag of Words (BoW) and TF-IDF are evaluated.

Models & Techniques

  • Gaussian Naive Bayes
  • Multinomial Naive Bayes
  • Bag of Words (BoW)
  • TF-IDF Vectorization
