Spam Email Detection

Machine Learning Project using Naive Bayes

Project Overview

Spam emails are a major issue in digital communication. This project uses Naive Bayes classifiers to automatically classify emails as Spam or Not Spam based on their content.

Dataset

The dataset contains labeled email messages and is preprocessed using text cleaning, tokenization, and vectorization techniques.

Models Used

  • Gaussian Naive Bayes
  • Multinomial Naive Bayes
  • Bag of Words (BoW)
  • TF-IDF Vectorization
